  - Kotlin

</br>

## 3. 핵심 기능
이 서비스의 핵심 기능은 무더위 쉼터로 지정된 곳을 지도 위에 표시해주는 것입니다.
기존 웹사이트에서는 주소를 입력해서 검색해야 하고, 검색 결과로 건물의 이름과 주소가 나옵니다.
하지만 이 앱을 사용하게 되면 검색할 필요 없이 사용자가 속한 행정구역 내에 있는 무더위 쉼터의 위치를 지도로 볼 수 있습니다.
